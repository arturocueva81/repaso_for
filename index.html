El Ciclo FOR

El ciclo FOR es un "robot" que repite una tarea por nosotros. Funciona bajo una estructura de 3 pasos: establece un inicio, define una condición de parada y actualiza el contador en cada vuelta.

💡 Estructura clave:

for (inicio ; condición ; actualizacion) { codigo }

- Inicio. Ejemplo:(let i = 0)
Es el punto de partida. Aquí creamos una variable (normalmente llamada i) que funciona como un contador para saber en qué vuelta vamos.

- Condición. Ejemplo: (i < 10)
Es la regla de oro. El ciclo seguirá funcionando mientras esto sea verdad. En cuanto la condición sea falsa, el robot se detiene.

- Actualización. Ejemplo: (i++)
Es lo que pasa al final de cada vuelta. Indicamos si el contador debe subir o bajar para avanzar a la siguiente repetición.

- Codigo. Es lo que queremos que el robot haga en cada vuelta. Puede ser cualquier tarea, como imprimir un mensaje o realizar cálculos. El robot seguirá haciendo esto una y otra vez hasta que la condición deje de ser verdadera.

En resumen, el ciclo FOR es como un asistente que hace el trabajo repetitivo por nosotros, siguiendo las instrucciones que le damos en su estructura.
